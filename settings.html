Configure Google Client ID

To use real Google SSO authentication on Vercel or localhost, you must create a Client ID in the Google Cloud Console.

1. Go to Google Cloud Console > APIs & Services > Credentials.
2. Create an OAuth Client ID for Web Application.
3. Add your current URL (e.g. https://hackathon-project-bice-tau.vercel.app) to Authorized JavaScript origins.
4. Copy and paste the Client ID below.
